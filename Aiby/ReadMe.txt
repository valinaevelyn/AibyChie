-- How to activate the virtual environment
1. Buka terminal VS Code
2. ketik "python -m venv venv" di terminal
3. ketik "cd venv\Scripts\activate" di terminal
Jika ada access error coba ketik
"Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process"
lalu "cd venv\Scripts\activate"
4. Lalu tekan "ctrl + shift + p", search “Python: Select Interpreter”. Lalu pilih venv yang telah dibuat

-- Run 
CTRL + ALT + N
